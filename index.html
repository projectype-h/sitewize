Protect Yourself Online

Sitewize is a smart browser extension that tells you whether a website is safe, suspicious, or dangerous using color-coded warnings.

How It Works

  • 🟢 Green: Safe Website
  • 🟡 Yellow: Suspicious Content
  • 🔴 Red: Dangerous/Malicious
  • ⚙️ Customizable alerts and popups

How to Install

  1. Download the extension ZIP.
  2. Unzip it to a folder.
  3. Open Chrome → chrome://extensions
  4. Turn on “Developer mode”.
  5. Click “Load unpacked” and choose the folder.
