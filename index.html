About Sitewize

Sitewize is your cyber safety companion. It checks every site you visit and shows color indicators: 🟢 Safe, 🟡 Warning, 🔴 Dangerous.

How It Works

  • 🟢 Shows a green light for safe websites
  • 🟡 Warns about risky links
  • 🔴 Blocks malicious pages
  • ⚙️ Includes settings for customization

Pricing

100% FREE — forever. No subscriptions. No sign-up.

Install the Extension

  1. Download the Sitewize extension ZIP file.
  2. Unzip it to a folder.
  3. Open Google Chrome, go to chrome://extensions
  4. Enable “Developer Mode” (top right).
  5. Click “Load unpacked” and select the folder you unzipped.
