Extension Features

🟢 Safe

Green icon when a site is clean and trusted.

🟡 Suspicious

Yellow alert for untrusted or strange behavior.

🔴 Dangerous

Red warning for phishing, scams, or malware.

⚙️ Settings

  • Customize icon colors
  • Enable/disable popup alerts
  • Whitelist trusted domains
  • Block known scam URLs

How to Install Sitewize

  1. Download the Sitewize extension ZIP
  2. Unzip the folder
  3. Open Chrome & visit chrome://extensions
  4. Enable Developer Mode
  5. Click “Load unpacked” and select the folder
