How It Works

🟢 Safe

Secure, trusted websites are marked green.

🟡 Suspicious

Websites with risky elements are flagged in yellow.

🔴 Dangerous

Harmful or malicious sites are clearly warned with red.

⚙️ Custom Settings

Customize alert styles and safety rules with ease.

How to Install

  1. Download the Sitewize ZIP extension file.
  2. Unzip it to your desktop or folder.
  3. Open Chrome and go to: chrome://extensions
  4. Enable Developer Mode.
  5. Click “Load unpacked” and select the unzipped folder.
